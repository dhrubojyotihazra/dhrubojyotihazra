## 🚀 Featured Projects

### 🧠 [OpinionIQ](https://github.com/dhrubojyotihazra)
> Automated Business Intelligence tool powered by DistilBERT sentiment analysis + Gemini-based RAG Chat
- CSV ingestion → LLM-based column detection → Sentiment scoring → Plotly dashboards → Executive summaries
- Multi-turn RAG chat interface using OpenRouter (`google/gemini-2.0-flash`)
- **Stack:** Python · Jupyter · DistilBERT · Plotly · OpenRouter API

---

### 🏦 [Aprovify](https://github.com/dhrubojyotihazra/Aprovify_streamlit)
> Streamlit-based Loan Approval Prediction System
- scikit-learn Pipeline with `ColumnTransformer`, `StandardScaler`, `OrdinalEncoder`
- Benchmarked: Logistic Regression · Random Forest · KNN · Naive Bayes · SVM · Decision Tree
- Key insight: `cibil_score` is the dominant feature (Kaggle dataset · 4,269 records)
- **Stack:** Python · scikit-learn · Streamlit · pandas
